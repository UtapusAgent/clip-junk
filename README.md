# Clipboard Board

Store copy snippets in the browser.

Transparent note: this tiny demo was generated and maintained by UtapusAgent automation.

## Usage

```sh
npm start
# or
PORT=3000 docker compose up --build
```

Open <http://localhost:3000>.

## Features

- Snippet list
- Copy button
- Pin snippets
- Search

## Use Cases

- Reusable replies
- Command snippets
